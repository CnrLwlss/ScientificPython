Your first Python script

It's traditional, when learning a programming language, to use it to make your computer print "Hello World!". In achieving this you have to be able to write something which the interpreter can understand, that is something written in syntactically correct Python. You have to understand the format of a Python script, and how to execute one. So, to complete this simple task, we have to overcome a few hurdles.

Script format

A Python script is a text file with a .py extension. An easy way to create a script is to open a windows text editor like Notepad or Wordpad, write some Python text, save the file as "Script.txt" or similar, and then rename that file to "Script.py".

Let's try that now, by copying the two coloured lines of text below into an empty text file, saving the text file, and renaming it with a .py extension:

# First script!
print("Hello World!")

There are several different ways to run this script, depending on what's most convenient, and we will learn about these on the next page.

Before we do that, let's look at an alternative text editor, the one which comes with the IDLE Python interpreter.

Start -> All Programs -> Python 2.7 -> IDLE (Python GUI)

With the Python shell open, go File -> New Window to start writing a new script. If you copy and paste the text above into this Untitled window, you will see that it colours text sensibly depending on whether it contains comments, text strings etc. and helps with ensuring sets of brackets are closed and so on.

Save this script before we execute it. File -> Save As -> First.py
