Installation

The first step on the way to using Python is to have the Python interpreter installed on your machine. The interpreter takes a Python script (a simple text file that you will write), checks that it can understand it (that it is written correctly in the Python language), and if so, it executes the instructions contained in the script. If the interpreter finds a language error (a syntax error), it helpfully tries to tell you about it and pinpoint the error so that you can correct it.

Installation difficulties specific to Newcastle University

Unfortunately, due to our university's security policy, it is no longer possible for students or staff to manually install Python onto university desktop machines. Generally, computing is much more straightforward if you have administrator rights on your machine. You can request administrator rights on a Newcastle University desktop machine to allow you to work more effectively, however the university default policy is not to allow this. In the absence of administrator rights, you can request that your institute's computer support team install Python (and any libraries required) onto your computer. Newcastle's IT department has expressed its full support for increasing the programming skills of biomedical students and will help. Please let me know if there are any problems. Python has been pre-installed on all of the machines in the Dene cluster (which is accessible by all) and should remain installed on those machines for the foreseeable future, however many useful libraries are currently missing.

Downloading Python

Microsoft Windows

We can download Python installers for Windows from the Python download page. Unless you have a specific reason not to (for instance, working with datasets > 2Gb in size) Windows users should install 32-bit Python (e.g. Windows Installer), regardless of whether you are using a 64-bit machine or a 32-bit machine.

A good alternative method for installing Python along with a wide range of useful scientific programming tools is to install the latest version of Google's pythonxy.

Linux & OSX

Python is installed by default on all versions of linux. You can check by opening a terminal window and typing python at the command line. It's best to just use the installed version if you can. Having multiple versions on one machine can be confusing.

Which version of Python?

There are two main versions of Python available: Python2.7.9 and Python3.4.0. We will use Python2.7.9 just because it has been around for longer and people have developed more supplementary packages for it. The language syntax is slightly different between Python2.x and Python3.x, and Python3.x will become the standard version in the future, but the differences are quite minor and only become apparent with advanced use.
