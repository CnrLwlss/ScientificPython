Python libraries

In the last example, you might have noticed a line which began with "import". One of the best things about Python is that many people have developed packages for it which add to its capabilities, typically by providing you with access to functions (which we will learn about in the Structure section). Many packages are included with the default Python installation (e.g. math, random, os, sys). Try importing math and typing "math.". You will get a prompt listing possible function names. To learn more, you could search online for the Python math package documentation, or type help(math) at the prompt in IDLE to find out more about the math package. This syntax works for any package. Try executing: 

import math
math.cos(math.pi)
To avoid typing "math" all the time, we could do the following (this is called importing a library into your namespace) which loads all the functionality in the math package as if it were a core part of Python: 
from math import *
cos(pi)

You should also try this out: 
import antigravity

Additional, external Python packages can be found online which can greatly extend Python's functionality. For example, try downloading and installing these two useful packages (Note you will need to request administrator permissions on most Newcastle University machines, but these should already be installed on computers in the Dene cluster):

  • Python Imaging Library
  • NumPy (Numerical Python)
